Terima kasih telah ingin berkontribusi ke perfume_pos!

Panduan singkat:
1. Fork repo dan buat branch fitur: `git checkout -b feat/deskripsi`
2. Jalankan: `cd backend && npm install`
3. Ikuti style: gunakan 2-spaces, semicolons optional
4. Tulis test untuk fungsionalitas baru
5. Buka PR ke branch `main` dengan deskripsi jelas dan link issue

Issues & PR:
- Gunakan template issue untuk bug/fitur.
- PR harus menyertakan langkah pengujian.

Kode etik dan perilaku dijelaskan di CODE_OF_CONDUCT.md.
